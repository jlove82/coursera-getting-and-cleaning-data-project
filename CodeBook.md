Code Book

This code book explains the result data fields of final project
(Coursers-Getting and Cleaning data project)


The dataset is located in the tidy_data.txt file of this repository.

If you want to check how to create and analyse the data set, Also check out
README.md file of this repository. 


Data

The tidy_data.xtx data file is a text file, including space-separated values.


Variables 

Each row have 79 averaged signal measurements based on identifiers (subject and activity)


Identifiers

subject
 integer ranges from 1 to 30 

activity 
 string 6 values 
 WALKING,WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING
